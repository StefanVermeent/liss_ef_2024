Vermeent et al. (in prep)
Preregistration Manuscript Supplement Data Scripts Analysis Objects


This repository contains data, code, and output for a manuscript with the working title “Attention shifting and inhibition abilities among people from adverse conditions”.

Do you want to download or clone the materials for this project? Go to https://github.com/stefanvermeent/liss_ef_2024

Directory Structure

The names of each folder are intended to be self-explanatory. There are eight top-level folders to organize the inputs and outputs of this project:

  1. Preregistration: The preregistration for the main analyses.
  2. Manuscript: The manuscript written in Quarto.
  3. Supplement: a supplemental text (to be submitted with the manuscript) documenting all secondary analyses in detail.
  4. Scripts: R-scripts that read, analyze, and produce all analysis objects.
  5. Data: Folder in which real LISS data can be placed to make the analyses fully reproducible. Note that we cannot openly share the raw data on the open repository.
  6. Analysis Objects: Folder containing all analysis objects.
  7. Codebooks: lists of variable names, labels, and value labels (where applicable).

Click on each of the folders to get more details.
